## install {#install}

1.  下载 Unity Hub
    <https://unity.cn/releases>
2.  在 Unity Hub 中主菜单 -&gt; installs -&gt; install Editor
3.  在安装好的 Editor 右侧的配置选择 -&gt; Add Modules
    安装不同平台的编译打包环境
4.  安装完报命令行错误，到安装根目录执行 `sudo chmod +x -R Editor/`,
    因为安装时很多文件的文件权限没有设置成可执行。
