## how to {#how-to}

install
: `sudo apt-get install hugo`

start
: 1.  `hugo new site <site_name>`
    2.  Add theme (`cd themes; git clone <theme-repo>`)
    3.  Add md file `hugo new posts/<x.md>`
    4.  Start server `hugo server -D`
